# Machine Learning Task 3 – Housing Price Prediction

This project was completed as part of my **Machine Learning Internship at Arch Technologies**.

The goal of this task was to build a machine learning model that predicts housing prices using the **California Housing Dataset**. The model analyzes various housing features such as income levels, number of rooms, population, and geographic location to estimate median house prices.

---

## 📊 Dataset

The dataset used in this project is the **California Housing Dataset** provided by Scikit-Learn.

Features included in the dataset:

- Median Income
- Average Rooms
- Average Bedrooms
- Population
- Households
- Latitude
- Longitude
- Average Occupancy

Target Variable:

- **Median House Value**

---

## ⚙️ Project Workflow

The following steps were performed to build the model:

1. Data Loading  
2. Data Exploration  
3. Data Cleaning  
4. Feature Engineering  
5. Train-Test Split  
6. Feature Scaling  
7. Model Training  
8. Model Evaluation  

---

## 🧠 Feature Engineering

Additional features were created to improve model performance:

- Rooms per Household  
- Bedrooms per Room  
- Population per Household  

These engineered features help the model better understand housing density and living conditions.

---

## 🤖 Model Used

The machine learning algorithm used for this project is:

**Random Forest Regressor**

Random Forest is an ensemble learning method that improves prediction accuracy by combining multiple decision trees.

---

## 📈 Model Evaluation

The model performance was evaluated using:

- **Mean Squared Error (MSE)**
- **R² Score**

Typical results obtained:
Mean Squared Error: 0.25350243892793284
R2 Score: 0.8065470832867492
